# react-select2-wrapper

Wrapper for [Select2](https://select2.github.io/)

## Installation

```
npm install react-select2-wrapper --save
```

## Documentation

[Guide](/guide)

## Development

### Webpack

- Run server
  ```
  npm run start
  ```

- Watch mode
  ```
  npm run watch
  ```

- Build for production
  ```
  npm run build
  ```
